About Convoy Mode

ETS2 Convoy Mode is a clean space for multiplayer trucking sessions. The current server runs as a Vanilla+ traffic setup, keeping the drive simple while still feeling more alive and polished.

Convoy features

  • Vanilla+ setup A lightweight experience that keeps the base ETS2 feel while adding enough extras to make the convoy more enjoyable.
  • Traffic sessions Built around road presence, spacing, realistic pacing, and smooth group driving.
  • EN / PL access Simple bilingual server naming so both English and Polish players know they are in the right place.

Session flow
